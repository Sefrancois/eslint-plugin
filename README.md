## 💡 What does it do ?

Provide a set of rules that you can use within [ESLint](https://eslint.org/).

## 🔧 Compatibility

| TypeScript | EcmaScript |
|------------|------------|
| \>= 2.8.0  | \>= ES2015 |

## 🎱 Dependencies

This package is dependencies-free.

## 💾 Installation

Nothing more than :

```shell
npm i -S @sefr/eslint-plugin
```
